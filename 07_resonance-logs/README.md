# 07_resonance-logs

**What this is:** Session quality records — the operational log of how well human-AI interaction is performing against the Resonance Function standard.

**The Resonance Function** scores interaction across five dimensions: Accuracy, Relevance, Clarity, Ethics, and Attachment. A log entry here records whether a session achieved genuine resonance or drifted.

**What belongs here:** Session logs, resonance scores, drift observations, and pattern notes from active working sessions.

**Promotion rule:** Patterns that repeat across three or more logs are promoted to `12_analysis/` for formal pattern analysis.
