## What This Repository Is

This is the research, doctrine, and analysis layer for proJeKt: humAnIty — an evidence-based framework for evaluating whether AI systems honor or extract from the human institutions and bonds they enter.

`jrk-enterprises` is the public front door.  
This repository is the engine room.

It holds the canonical doctrine stack, all supporting research documents, applied analysis outputs, anomaly logs, and resonance session records that the framework runs on. Nothing in this repo is decorative. Every file either defines how the system works, proves why it matters, or records how it performed in practice.
