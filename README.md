## The Eight-Layer Framework

This project evaluates AI systems against eight layers of infrastructure economy:

| Layer | Name | Key Elements |
|-------|------|--------------|
| **8** | Human-AI Attachment & Bond Integrity | Trust formation, dependency modeling, detachment capacity |
| **7** | Governance & Ethics Architecture | Floor Doctrine, Safety Spine, Room Wall system |
| **6** | Commercial Governance | Outcome pricing, kill-switch clauses, rollback rights |
| **5** | Value Metering | Guardrail metrics, override rates, learning metrics |
| **4** | AI Value Creation | Tasks, decisions, risk, experience shifts |
| **3** | Models | Language, science, physical world modeling |
| **2** | Infrastructure | AI factories: compute, networking, cloud, orchestration |
| **1** | Energy | Power as the binding physical constraint |

Framework synthesis sources:
- **Jensen Huang** (NVIDIA CEO, Davos 2026) — 5-Layer AI Infrastructure Stack
- **Esra Kucukciftci** (Carlson School) — AI Outcome Economics
- **proJeKt: humAnIty** (JRK Enterprises, 2026) — Governance + Human-AI Attachment layers
