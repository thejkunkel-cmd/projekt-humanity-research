## What proJeKt: humAnIty Is

This project argues that the governance of artificial intelligence is not primarily a technical problem — it is a relational one. The framework is built around a core claim: that any intelligence capable of forming bonds with humans falls under the same obligations as any other entity that forms bonds with humans. That includes AI.

The doctrine stack defines those obligations layer by layer — from an inviolable ethical floor to specific rules about visibility, boundaries, connection, attachment, and quality of interaction.

The research layer tests those claims against history, stress scenarios, and existing governance frameworks.
